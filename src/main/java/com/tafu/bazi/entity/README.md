# 实体目录 (entity)

## 目录用途
存放 JPA 实体类 (Entity)，负责映射数据库表结构。

## 内容清单
| 文件/目录 | 类型 | 用途说明 |
|:--- |:--- |:--- |
| `PaymentOrder.java` | Class | 支付订单 (payment_orders 表) |
| `PointsAccount.java` | Class | 积分账户 (points_accounts 表) |
| `PointsPackage.java` | Class | 充值套餐 (points_packages 表) |
| `PointsTransaction.java` | Class | 积分流水 (points_transactions 表) |
| `Subject.java` | Class | 测算对象 (subjects 表) |
| `Task.java` | Class | 异步任务 (tasks 表) |
| `ThemeAnalysis.java` | Class | 主题分析结果 (theme_analyses 表) |
| `User.java` | Class | 用户实体 (users 表) |
| `VerificationCode.java` | Class | 验证码 (verification_codes 表) |

## 维护说明
当这个文件/文件夹发生改动时，同步改动说明文件以及上一层文件夹对本文件/文件夹的描述。
